# React Hooks App - IS093A Semana 07

## Instalación

```bash
npm install
npm run dev
```

## Estructura

```
src/
  context/   → AppContext.jsx (useContext - ThemeContext, AuthContext)
  hooks/     → useFetch.js, useForm.js (Hooks Personalizados)
  components/→ Header, Counter, PostList, ContactForm
```
